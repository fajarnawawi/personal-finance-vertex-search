# Finance Brain

A personal finance knowledge base over Telegram. Ingests podcasts, PDFs, articles, voice memos, and receipt photos — indexes everything in Vertex AI Search — answers questions with grounded citations.

**Primary interface:** Telegram bot at `bot.askfajar.com`
**Stack:** Python 3.11 · Cloud Run · Vertex AI Search · AWS Transcribe · BigQuery · Google Tag Gateway

## Quick start

Send a message to the bot and ask anything about your indexed content. Send a receipt photo to log it. Send a voice memo to transcribe and index it.
